## 프로젝트 목표
- 유튜브 페이지를 비슷하게 구현해보기
- 주어진 api를 활용하여 화면에 구현하고자 하는 방식대로 나타내기
- 팀 프로젝트를 진행하면서 깃허브, 노션등의 협업도구를 활용하여 익숙해지기
<br>


## 프로젝트 진행방식
* html / css / js 3가지의 기술스택만을 사용하여 유튜브 메인페이지, 채널페이지, 영상페이지를 기획하고 구현.
* 릴레이 코딩 방식을 통해 프로젝트 진행.
* 릴레이 코딩이란 매일 다른 브랜치에서 코딩진행. (하루는 html, 다음 하루는 css, 다음 하루는 js 브랜치에서 작업하는 새로운 방식)
<br>

## 핵심 기능

#### 메인페이지
- 영상 목록 표시
- 사이드바 스크롤 및 호버 효과
- 사이드바 '더보기' 기능
- 사이드바 크기 조절 토글
- 헤더 아이콘 설명 텍스트 호버 효과
- 수평 스크롤 가능한 nav바
- 대소문자 구분 없는 검색 기능 (클릭 및 엔터)

#### 채널페이지
- 채널 페이지 접근 및 대표 영상 자동재생
- 구독 기능 및 사이드바 업데이트
- 사이드바 토글 기능
- 채널 페이지 내 검색 → 메인페이지 결과 출력
- 채널 내부 검색 기능

#### 비디오페이지
- 영상 선택 및 채널 프로필 및 추천 영상 표시
- 사이드바 토글 기능
- 사이드바 활성 시 화면 불투명 효과
- 댓글 기능 (저장, 수정, 삭제)
- 추천 영상 클릭 시 페이지 이동
- 유튜브 로고 클릭 시 홈페이지로 이동
