## 프로젝트 진행방식
* html / css / js 3가지의 기술스택만을 사용하여 유튜브 메인페이지, 채널페이지, 영상페이지를 기획하고 구현.
* 릴레이 코딩 방식을 통해 프로젝트 진행.
* 릴레이 코딩이란 매일 다른 브랜치에서 코딩진행. (하루는 html, 다음 하루는 css, 다음 하루는 js 브랜치에서 작업하는 새로운 방식)
<br>
